# 参考论文与相关工作

  https://arxiv.org/abs/2511.17190

  https://arxiv.org/abs/2512.17053  

- Talaei *et al.* **CHESS: Contextual Harnessing for Efficient SQL Synthesis.** arXiv:2405.19555, 2024.  
  https://arxiv.org/abs/2405.19555
